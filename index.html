Projet de simulation en Javascript : Simulation de la gravité

Auteurs

Prénom Nom
Wassim Saïdane
Raphaël Marraccini

Description

Notre projet conssiste à réaliser une application web qui simule la gravité sur différentes planètes. Ainsi nous allons simuler la chute d'objets sur le sol. Grâce aux calculs que nous allons retranscrire en javascript nous pourions créer une implication simple, intuitive et interactive.

Etape

  1. Trouver les bibliothèque utiles pour réaliser la simmulation et le moteur graphique
  2. Faire les calculs scientifiques nécessaires
  3. Rajouter une interface intuitive
